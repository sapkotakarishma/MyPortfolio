Natural Language Processing (NLP)

NLP helps computers understand and work with human language: text, documents, and conversations.

1. Text Preprocessing

Cleaning raw text before modeling.

import re

text = "Hello!!! This is NLP 101..."
clean = re.sub(r"[^a-zA-Z ]", "", text).lower().strip()
print(clean)

2. Tokenization

Splitting text into words or sentences.

from nltk.tokenize import word_tokenize

tokens = word_tokenize("NLP makes text understandable.")
print(tokens)

3. POS Tagging & Entities (spaCy)

Understanding grammar and extracting names, places, organizations.

import spacy
nlp = spacy.load("en_core_web_sm")

doc = nlp("Apple is buying a UK startup.")

for ent in doc.ents:
    print(ent.text, ent.label_)

4. TF‑IDF & Transformers

From classic vectorization to modern transformer models.

from sklearn.feature_extraction.text import TfidfVectorizer
from transformers import pipeline

docs = ["I love NLP", "NLP is fun"]
tfidf = TfidfVectorizer()
X = tfidf.fit_transform(docs)

classifier = pipeline("sentiment-analysis")
print(classifier("This portfolio looks amazing!"))

SQL (Structured Query Language)

SQL is used to extract, clean, and analyze data stored in relational databases.

1. Basic Queries

SELECT, WHERE, ORDER BY.

SELECT name, age, diagnosis
FROM patients
WHERE age > 50
ORDER BY age DESC;

2. Joins

Combine data from multiple tables.

SELECT p.name, v.visit_date
FROM patients p
JOIN visits v
ON p.patient_id = v.patient_id;

3. Aggregations & Group By

Summaries by category.

SELECT diagnosis, COUNT(*) AS count
FROM patients
GROUP BY diagnosis
ORDER BY count DESC;

4. Window Functions

Advanced analytics: ranking, running totals.

SELECT 
    name,
    age,
    RANK() OVER (ORDER BY age DESC) AS age_rank
FROM patients;
