Scikit‑learn

Machine learning models for classification & regression.

from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

X = df.drop("target", axis=1)
y = df["target"]

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = RandomForestClassifier()
model.fit(X_train, y_train)
model.score(X_test, y_test)

SpaCy – POS Tagging & Named Entities

Extracts grammar structure and identifies people, places, and organizations.

import spacy
nlp = spacy.load("en_core_web_sm")

doc = nlp("Apple is looking at buying a UK startup.")

for ent in doc.ents:
    print(ent.text, ent.label_)

Sentiment Analysis

Classifies text as positive, negative, or neutral.

from nltk.sentiment import SentimentIntensityAnalyzer

sia = SentimentIntensityAnalyzer()
print(sia.polarity_scores("I love data science!"))

TF‑IDF Vectorization

Converts text into numerical features for ML models.

from sklearn.feature_extraction.text import TfidfVectorizer

docs = ["I love NLP", "NLP is fun", "I enjoy machine learning"]
tfidf = TfidfVectorizer()
X = tfidf.fit_transform(docs)

print(tfidf.get_feature_names_out())

Transformers (HuggingFace)

State‑of‑the‑art models for classification, QA, and embeddings.

from transformers import pipeline

Word Embeddings (Word2Vec)

Vector representations of words capturing meaning & similarity.

from gensim.models import Word2Vec

sentences = [["nlp","is","fun"],["machine","learning","is","powerful"]]
model = Word2Vec(sentences, vector_size=50, min_count=1)

print(model.wv["nlp"])
