Natural Language Processing (NLP)

NLP lets me work with unstructured text—clinical notes, social media, survey responses—and turn it into signals.

1. Text Preprocessing

Cleaning raw text is essential before modeling or feature extraction.

import re

text = "Hello!!! This is NLP 101..."
clean = re.sub(r"[^a-zA-Z ]", "", text).lower().strip()
print(clean)

2. Tokenization

Tokenization splits text into words or sentences, which becomes the basis for further analysis.

from nltk.tokenize import word_tokenize

tokens = word_tokenize("NLP makes text understandable.")
print(tokens)

3. POS Tagging & Entities (spaCy)

Part‑of‑speech tags and named entities help capture structure and meaning in text.

import spacy
nlp = spacy.load("en_core_web_sm")

doc = nlp("Apple is buying a UK startup.")

for ent in doc.ents:
    print(ent.text, ent.label_)

4. TF‑IDF & Transformers

I use both classic vectorization and modern transformer models, depending on the problem and constraints.

from sklearn.feature_extraction.text import TfidfVectorizer
from transformers import pipeline

docs = ["I love NLP", "NLP is fun"]
tfidf = TfidfVectorizer()
X = tfidf.fit_transform(docs)

classifier = pipeline("sentiment-analysis")
print(classifier("This portfolio looks amazing!"))

SQL (Structured Query Language)

SQL is how I access, filter, and aggregate data stored in relational databases.

1. Basic Queries

SELECT, WHERE, and ORDER BY form the backbone of most analytical queries.

SELECT name, age, diagnosis
FROM patients
WHERE age > 50
ORDER BY age DESC;

2. Joins

Joins let me combine data from multiple tables into a single, coherent view.

SELECT p.name, v.visit_date
FROM patients p
JOIN visits v
ON p.patient_id = v.patient_id;

3. Aggregations & Group By

Aggregations summarize data across groups—critical for reporting and monitoring.

SELECT diagnosis, COUNT(*) AS count
FROM patients
GROUP BY diagnosis
ORDER BY count DESC;

4. Window Functions

Window functions enable ranking, running totals, and more advanced analytics directly in SQL.

SELECT 
    name,
    age,
    RANK() OVER (ORDER BY age DESC) AS age_rank
FROM patients;

Statistics

2. Hypothesis Testing

Hypothesis tests help compare groups and quantify whether differences are likely due to chance.

from scipy import stats

group1 = df[df["group"]=="A"]["value"]
group2 = df[df["group"]=="B"]["value"]

stats.ttest_ind(group1, group2)
